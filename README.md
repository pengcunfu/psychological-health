# 心理健康平台

## 项目概述

心理健康平台是一个综合性的心理健康服务系统，旨在为用户提供专业的心理咨询、课程学习和心理健康资源。该平台由三个主要部分组成：

1. **微信小程序端**：面向普通用户，提供心理咨询预约、课程学习、心理测评等服务
2. **PC管理后台**：面向管理员和咨询师，提供系统管理、数据分析、内容管理等功能
3. **服务端**：为前端提供API接口，处理业务逻辑，管理数据存储
